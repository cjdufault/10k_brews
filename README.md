# 10k_brews

Hosted at: [cjdufault.pythonanywhere.com](https://cjdufault.pythonanywhere.com)

10,000 Brews is a database of breweries, wineries, distilleries, and cideries located in Minnesota. It allows a user to keep track of which places they've visited and what drinks they've tried.

## Requirements
- Requires all packages in requirements.txt -- can be installed with pip
- Requires a [MapBox](https://mapbox.com) API token
